Desktop Notes

A Desktop Notes egy PC-kre és laptopokra készített alkalmazás. Megjeleníthetjük és rendszerezhetjük vele jegyzeteinket PC-nken vagy laptopunkon. A program késõbb támogatni fogja a jegyzetek kinyomtatását is.

A Desktop Notes-ban megváltoztatható a felület nyelve, és könnyedén lehet további nyelvekkel bõvíteni.

A Desktop Notes egy Java alkalmazás, így használhatjuk Linuxon, Machintoson, még Windows alatt is.

Az alkalmazásról készített képek megtekinthetõk a honlapon.
A program teszt változata már letölthetõ a Letöltések menüpontban.
Letölthetõ a jar fájl is önmagában, amit a Java futtató környezet el tud indítani (pl. Machintoson). Ha ezt használjuk, elsõ futtatáskor fogja letölteni a futás szükséges további állományokat, ezért szüksége van Internet kapcsolatra.
Machintossal kapcsolatban nincsenek tapasztalataim. Hogyha bárki használja a programot Machintoson, szívesen fogadok egy üzenetet hogy megfelelõen mûködik-e, esetleg pár képet a programról futás közben (Kapcsolat).
Ha tud valaki készíteni Machintos telepítõt hozzá, hálás leszek érte, és az is felkerül az oldalra. (Természetesen az illetõ nevének feltüntetésével).
