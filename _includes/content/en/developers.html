Information for developers

Here are information for everyone who wants to write extra applications for the program or wants to use the notes in his programs.

  • The format of the Pocket Notes files

    The internal structure of the .apn files. If you want to write your own reader or writer, you can do it based on this document.

  • Pocket Notes file handler PHP library

    A PHP library, which you can work with .apn files with from a PHP script. The on-line converter also uses this library.
